# Servicio-de-Incidencias
Es el microservicio principal del sistema. Gestiona el ciclo de vida del reporte y consume la API del  clima para sugerencias operativas (Si la API reporta "Tormenta" o "Lluvia intensa": El sistema  muestra una alerta, por ejemplo: "No se recomienda enviar cuadrillas para reparación de baches o  configuración de semáforos debido al clima") .
